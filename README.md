#PHPDeveloperParte01
Solution to the first part of the PHPDeveloper exam.

##ChangeString.php
Usage
```bash
 echo "entrada : **Casa 52Z" . "\n";
 echo " salida :" . build("**Casa 52Z");
``` 
 
##CompleteRange.php
Usage
```bash
echo "entrada : 55,58,60" . "\n";
echo " salida :" . build("55,58,60");
``` 

##ClearPar.php
Usage
```bash
echo "entrada : ()(()" . "\n";
echo " salida :" . build("()(()");
```
